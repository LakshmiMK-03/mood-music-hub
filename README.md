# Mood Music Hub

AI-powered emotion detection and personalized music recommendation system for stress management.

## Project Info

This project uses a **Python Flask** backend with a lightweight **HTML/CSS/JavaScript** frontend.

## How to Run

### Prerequisites
- Python 3.x
- pip

### Steps

1.  **Clone/Navigate to the directory**:
    ```bash
    cd mood-music-hub
    ```

2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Application**:
    ```bash
    python app.py
    ```

4.  **Open in Browser**:
    Visit [http://localhost:5000](http://localhost:5000)

## Features

- **Emotion Analysis**: Analyzes text, voice, (simulated) and images to detect mood.
- **Music Recommendations**: Generates playlists based on your emotional state.
- **Relaxation Tips**: Guided breathing exercises and mindfulness tips.

## Technologies

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Custom CSS with variables (No external CSS frameworks required)
- **Icons**: Lucide Icons
