Project Abstract

“In today’s fast-paced world, stress and emotional imbalance have become major concerns affecting mental health and productivity. Many individuals struggle to express their emotions effectively, leading to anxiety, depression, and reduced focus.

Music has proven to be an effective tool for emotional regulation and stress management. This project presents an AI-based web application that detects user emotions and stress levels using text analysis, voice recognition, and facial expression detection.

Based on the detected emotional state, the system recommends personalized music playlists and provides relaxation tips such as breathing exercises, mindfulness suggestions, and stress-relief activities.”

How It Works

Text Input

Text is processed using NLP techniques — Tokenization, Stop-word Removal, Lowercasing, and TF-IDF Feature Extraction — then classified using ML algorithms.

Voice Input

Voice input is converted into text using Speech-to-Text technology and then analyzed using the same NLP + ML pipeline for emotion classification.

Image Input

Image input is processed using OpenCV for face detection and a Convolutional Neural Network (CNN) model for facial emotion recognition.

Emotion & Stress Detection

ML classification detects emotions — Happy, Sad, Angry, Neutral — and predicts stress levels as Low, Medium, or High.

Music Recommendation

Based on detected emotional state, the system maps results to predefined music playlists for emotional regulation and stress relief.

Relaxation Tips

When medium or high stress is detected, the system provides breathing exercises, mindfulness suggestions, and break reminders.

Admin Dashboard

Admin can monitor user activity, analysis logs, recommendation history, and system performance for better management and system maintenance.
