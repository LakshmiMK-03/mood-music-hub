Project Abstract

In today's fast-paced world, stress and emotional imbalance are growing concerns. Many individuals are unable to openly express their emotions, which may lead to anxiety, depression, and reduced productivity. Music has proven to be an effective tool for emotional regulation and stress reduction. This system is a web-based application designed to detect user emotions and stress levels and provide personalized music recommendations along with relaxation tips.

How It Works

Text Input

Text is processed using NLP techniques — Tokenization, Stop-word Removal, Lowercasing, and TF-IDF Feature Extraction — then classified using ML algorithms.

Voice Input

Voice input is converted into text using Speech-to-Text technology and then analyzed using the same NLP + ML pipeline for emotion classification.

Image Input

Image input is processed using OpenCV for face detection and a Convolutional Neural Network (CNN) model for facial emotion recognition.

Emotion & Stress Detection

ML classification detects emotions — Happy, Sad, Angry, Neutral — and predicts stress levels as Low, Medium, or High.

Music Recommendation

Based on detected emotional state, the system maps results to predefined music playlists for emotional regulation and stress relief.

Relaxation Tips

When medium or high stress is detected, the system provides breathing exercises, mindfulness suggestions, and break reminders.
